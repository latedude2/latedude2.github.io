Party LEDs

A personal project for the Physical Interface Design course at Aalborg university. I created a party lighting system using an LED strip, Arduino and a Processing script. The Processing script analyses the sound made by the computer and sends an appropriate signal to the Arduino. The Arduino then takes user input into account and lights up the LED strip accordingly.

Technologies used: Arduino, Processing.
