Wildfire Rush

A game developed by me and 4 other students during a game development course at Aalborg university. For this project, I acted as the team lead, programmer and game designer. We won the most engaging game title out of 19 groups of students that made games the same semester at AAU! The levels were loaded during runtime from a website I setup which hosted the level files. This was done to easier edit the levels when using the WebGL build.

Technologies used: C#, Unity, JavaScript.

I worked with the Unity UI system to setup the menu layout so it adapted to various screen sizes.

I used the Unity Particle System to create a fire extinguisher effect.

I used Cinemachine to capture all the shots you can see in the trailer of this game.
